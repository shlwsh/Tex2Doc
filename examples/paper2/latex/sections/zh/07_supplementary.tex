\paragraph{一键复现。}
在仓库根目录执行 \texttt{bash scripts/paper1\_run\_tcm\_full.sh}，将依次完成：导入划分、训练 B3、ROI 标定 $\tau$、主矩阵（3 seeds）、图表与 PDF 生成。

\paragraph{工件清单。}
\begin{itemize}
  \item \texttt{tcm\_\{train,val,test\}.json}：共 6719 张，含 COCO \texttt{bboxes}；
  \item \texttt{recommended\_tau.json}：ROI 标定（$n{=}1144$）；
  \item \texttt{roi\_ablation.json}：全图 vs.\ 舌区 ROI；
  \item \texttt{b3\_mobilenet.pt}、\texttt{b3\_training\_meta.json}：B3 权重与验证准确率；
  \item \texttt{calibration\_val.csv}、\texttt{main\_seed*\_detail.csv}：分数与帧级日志。
\end{itemize}

\paragraph{环境与超参数。}
Edge-IQA 仅需 NumPy/Pillow；B3 训练使用 \texttt{doctor/paper1/.venv}（PyTorch）。
WSL 下数据集路径见 \texttt{tcm\_paths.yaml}。
Edge-IQA：512 缩放，清晰度/曝光权重 0.65/0.35；ROI padding 8\%。
B3：MobileNetV3-Small，224 输入，AdamW $10^{-3}$，5 epoch，batch 64。
路由：$\tau$ 来自验证集中位数，$K{=}2$。

\paragraph{JSONL 字段。}
\texttt{trial\_id}、\texttt{image\_path}、\texttt{q\_img}、\texttt{flags}、\texttt{retry\_count}、\texttt{route\_decision}、\texttt{latency\_ms}、\texttt{t\_capture}、\texttt{t\_iqa}、\texttt{t\_route}。

\paragraph{局限与伦理。}
合成高斯模糊；运行时需检测框或检测器；B3 与 Edge-IQA 的 $\tau$ 标度不一致；M6 不与表 II 直接对比。
使用公开 ShezhenV3-COCO；JSONL 不含患者标识。

\paragraph{未来工作与摘要表。}
端侧舌检测 ROI；混合 Edge-IQA+B3 头；基于 FR3 日志的运动模糊增广。
表 S1：全图 $\tau{=}0.498$，ROI $\tau{=}0.465$（详见 \texttt{roi\_ablation.json}）。
表 S2：5594 训练图，11188 对，验证准确率 98.3\%。
