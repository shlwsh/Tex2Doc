On the validation split ($n{=}572$ clear + synthetic blur pairs), Fig.~\ref{fig:iqa-hist} shows overlapping $Q_{\mathrm{img}}$ histograms.
Table~\ref{tab:iqa-calibration} lists cohort medians ($0.516/0.414$), deployed $\tau_{B2}{=}0.465$, and negative separability ($-0.541$): some blurred validation crops score above the weakest clear crops, so $\tau$ is a cohort-median operating point rather than a hard separator.
Spearman $\rho{\approx}0.36$--$0.47$ on real texture motivates pairing interpretable B2 with learned comparators (Table~\ref{tab:learned-hybrid}).
