\chapter{Overview}
This report-style fixture validates chapter headings, lists, display math, and bibliography commands under the generic fallback profile.
\section{System Model}
The system receives a source bundle $B$ and emits a document graph $G$.
\begin{equation}
  G = R(P(B), M) \label{eq:graph-render}
\end{equation}
\section{Checklist}
\begin{itemize}
  \item detect a generic profile;
  \item preserve section hierarchy;
  \item keep references readable.
\end{itemize}
