## 核心功能详解

### 1. LaTeX 转 Word 转换

#### 转换模式

| 模式 | 说明 | 适用场景 | 是否需要登录 |
|------|------|----------|--------------|
| **本地（WASM）** | 使用 WebAssembly 在浏览器本地转换，文件不离开设备 | 简单文档、隐私敏感内容 | 否 |
| **云端** | 上传至 Tex2Doc API 服务器处理 | 复杂文档、大型项目 | 是 |

#### 配置文件（Profile）

| 配置 | 说明 | 适用文档类型 |
|------|------|--------------|
| **Standard** | 标准转换配置 | 通用文档、报告 |
| **Academic** | 学术文档优化 | 论文、学位论文 |
| **Publication** | 出版级质量 | 期刊文章、书籍章节 |

#### 质量等级（Quality）

| 等级 | 说明 | 处理速度 | 输出质量 |
|------|------|----------|----------|
| **Preview** | 预览模式 | 最快 | 基础格式 |
| **Balanced** | 平衡模式 | 适中 | 良好格式 |
| **Strict** | 严格模式 | 较慢 | 最佳格式 |

## 快速开始

### 方式一：弹出窗口转换

1. 点击浏览器工具栏中的 Tex2Doc 图标
2. 点击选择文件区域，选择 LaTeX 项目的 ZIP 文件
3. 如果主文件不是 `main.tex`，修改主文件名
4. 选择转换模式（本地/云端）
5. 如使用云端，可选择配置文件和质量等级
6. 点击「Convert」开始转换
7. 转换完成后自动下载 DOCX 文件

### 方式二：网站集成转换

**Overleaf 项目**
1. 打开 Overleaf 项目页面
2. 页面右下角出现 Tex2Doc 悬浮按钮
3. 点击按钮，一键转换

**arXiv 论文**
1. 打开 arXiv 论文摘要页
2. 在下载按钮区域出现 Tex2Doc 按钮
3. 点击「Download & Convert with Tex2Doc」

## 完整操作指南

### 文件上传

**支持的文件格式**
- ZIP 压缩包（包含 LaTeX 源文件）
- 包含 .tex 文件和依赖资源

**上传步骤**
1. 点击虚线框区域或拖拽文件到该区域
2. 选择包含 LaTeX 项目的 ZIP 文件
3. 确认文件名和大小显示正确

**文件要求**
- 文件大小建议小于 10MB（本地模式）
- ZIP 包内应包含完整的 LaTeX 项目结构
- 包含一个主 .tex 文件

### 主文件配置

**默认设置**
- 主文件默认为 `main.tex`

**自定义主文件**
- 如果主文件不是 `main.tex`，在「Main TeX file」输入框中修改
- 例如：`thesis.tex`、`article.tex`、`report.tex`

### 本地转换（WASM）

**特点**
- 文件在本地浏览器处理，不上传到服务器
- 无需登录账户
- 适合简单文档

**操作步骤**
1. 选择「Local (WASM)」模式
2. 上传 ZIP 文件
3. 设置主文件名
4. 点击「Convert」
5. 等待转换完成

**限制**
- 不支持复杂的 LaTeX 包和模板
- 文件大小限制 10MB
- 无转换质量报告

### 云端转换

**特点**
- 上传至云端处理，支持复杂文档
- 支持多种配置文件和质量等级
- 提供转换质量报告
- 需要登录账户

**操作步骤**
1. 登录账户（点击 Sign In）
2. 选择「Cloud」模式
3. 上传 ZIP 文件
4. 设置主文件名
5. 选择配置文件（Standard / Academic / Publication）
6. 选择质量等级（Preview / Balanced / Strict）
7. 点击「Convert」
8. 等待转换完成并自动下载

### 查看转换状态

**任务状态**
| 状态 | 说明 |
|------|------|
| Pending | 等待中 |
| Processing | 处理中 |
| Completed | 已完成 |
| Failed | 失败 |
| Expired | 已过期 |

**查看历史任务**
1. 点击扩展弹出窗口底部的「Jobs」
2. 查看最近 3 次转换记录
3. 点击可查看详细信息

### 侧边面板（Chrome/Edge 专属）

侧边面板提供 SaaS 风格的全功能管理界面，访问方式：

1. **Chrome**：右键点击扩展图标 → 选择「打开侧边面板」或点击工具栏中的侧边面板按钮
2. **Edge**：点击扩展图标旁的侧边面板按钮

**侧边面板功能**
- **Jobs 任务**：查看完整转换历史记录、下载结果、查看质量报告
- **Billing 计费**：查看套餐、订阅升级、兑换码充值
- **Feedback 反馈**：提交问题反馈、导出诊断日志
- **Account 账户**：账户信息、用量统计、续期提示

### 诊断日志导出

当遇到问题时，可以导出诊断日志用于问题排查：

1. 打开侧边面板
2. 切换到「Feedback」标签
3. 点击「Export Diagnostics」按钮
4. 下载 JSON 文件并附在反馈中

## 高级功能

### 右键菜单

扩展注册了浏览器右键菜单：

| 菜单项 | 功能 |
|--------|------|
| **Open Tex2Doc** | 打开扩展弹出窗口 |

**使用方式**
1. 在任意页面右键点击
2. 选择「Open Tex2Doc」
3. 打开扩展进行转换

### 任务历史记录

扩展会自动保存转换任务记录：

- 最近任务自动显示在弹出窗口
- 任务状态实时更新
- 支持查看历史转换结果

**清除历史**
- 在设置页面清除所有任务记录

### 转换结果通知

转换完成时会发送浏览器通知：

**成功通知**
- 显示「Conversion completed!」
- 可直接点击下载文件

**失败通知**
- 显示错误原因
- 建议重试或联系客服

**登录步骤**
1. 点击扩展图标打开弹出窗口

## 故障排除

### 常见问题

#### 1. 扩展图标不显示

**原因**
- 扩展未正确加载

**解决**
1. 打开 `chrome://extensions/`
2. 确认 Tex2Doc 扩展已启用
3. 点击「重新加载」按钮
4. 如仍不显示，卸载后重新加载

#### 2. 云端转换按钮不可用

**原因**
- 未登录账户

**解决**
1. 点击「Sign In」登录
2. 如无账户，点击注册
3. 登录后按钮自动启用

#### 3. 转换失败

**常见原因**
- LaTeX 语法错误
- 缺少必要的依赖包
- 文件过大
- 网络连接问题
- 配额已用尽

**解决步骤**
1. 确认 LaTeX 文件语法正确
2. 尝试使用本地模式转换
3. 检查网络连接
4. 确认账户配额充足
5. 查看错误信息并重试

#### 4. 文件上传失败

**原因**
- 文件格式不正确
- 文件过大
- 网络问题

**解决**
1. 确认上传的是 ZIP 文件
2. 检查文件大小（本地模式限制 10MB）
3. 重试上传

#### 5. Overleaf 按钮不显示

**原因**
- 不在项目页面
- 页面未完全加载
- 内容脚本未执行

**解决**
1. 确认在 Overleaf 项目页面
2. 等待页面完全加载
3. 刷新页面重试

#### 6. arXiv 按钮不显示

**原因**
- 不在摘要页
- 页面结构变更
- 内容脚本冲突

**解决**
1. 确认在 arXiv 摘要页（/abs/ 路径）
2. 刷新页面
3. 如仍不显示，检查是否有其他扩展冲突

#### 7. 登录失败

**常见原因**
- 邮箱或密码错误
- 账户不存在
- 服务器错误

**解决**
1. 确认邮箱密码正确
2. 检查大小写
3. 尝试重置密码
4. 联系客服

#### 8. 配额显示不正确

**原因**
- 数据未同步
- 会话过期

**解决**
1. 刷新扩展
2. 重新登录
3. 等待自动同步

### 错误代码

| 错误码 | 说明 | 解决方案 |
|--------|------|----------|
| `AUTH_ERROR` | 认证失败 | 重新登录 |
| `QUOTA_EXCEEDED` | 配额用尽 | 升级套餐或等待重置 |
| `CONVERSION_FAILED` | 转换失败 | 检查 LaTeX 语法 |
| `FILE_TOO_LARGE` | 文件过大 | 减小文件大小 |
| `INVALID_FILE` | 文件格式错误 | 上传 ZIP 文件 |
| `WASM_LOAD_FAILED` | WASM 加载失败 | 刷新页面重试 |
| `SESSION_EXPIRED` | 会话过期 | 重新登录 |
| `NETWORK_ERROR` | 网络错误 | 检查网络连接 |

### 获取帮助

**反馈问题**
1. 点击设置中的「Submit Feedback」
2. 选择问题类型（Issue / Requirement / Other）
3. 填写标题和描述
4. 可附上转换任务 ID
5. 提交反馈

**联系客服**
- 通过扩展内的反馈功能提交工单
- 或访问 Tex2Doc 官网获取支持
