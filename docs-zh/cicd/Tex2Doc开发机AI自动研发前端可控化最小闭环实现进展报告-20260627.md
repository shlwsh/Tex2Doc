## 3. 状态机

```
submitted → triaged → needs_approval → queued_for_dev → claimed → coding
                                                              ↓
                                                        local_validating
                                                              ↓
                                                        local_failed (可重试)
                                                              ↓
                                                          pr_open → ci_running
                                                              ↓
                                                          ci_failed (可重试)
                                                              ↓
                                                        ready_for_merge → production_deployed → notified

needs_approval → rejected (拒绝)
needs_approval → needs_human (转人工)
```
